## 🚀 About The Project

The **Church Management System** is a powerful, modern, and fully responsive web application built to automate church administration and eliminate manual tracking problems.

This platform helps churches manage members, birthdays, anniversaries, and important events with **zero confusion and maximum efficiency**.

✅ Smart  
✅ Fast  
✅ Reliable  
✅ User-Friendly  

---

## ✨ Key Features

- 🎂 Today's Birthday & Anniversary Highlight  
- 📅 Tomorrow Reminder System  
- 🗓️ Next 7 Days Upcoming Events  
- ❗ Missed Wishes Detection  
- 👨‍👩‍👧 Member Database Management  
- 🔐 Secure Admin Panel  
- ⚡ Lightning Fast UI  
- 🎨 Modern Responsive Design

# 🔽 Clone repository
git clone https://github.com/Rishidevlx/Church-management-System.git

# 📂 Enter project folder
cd Church-management-System

# 📦 Install dependencies
pip install -r requirements.txt

# ▶️ Run the app
python app.py
